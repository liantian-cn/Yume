介绍

这是一个使用Python将多个Markdown文件合并生成单页HTML的工具。

功能特点

  • 支持多个Markdown文件按顺序合并
  • 使用Jinja2模板引擎渲染
  • 自动生成目录
  • 美观的TerminalCSS样式

使用方法

  1. 在content目录下添加Markdown文件
  2. 运行main.py脚本
  3. 在output目录查看生成的HTML文件

使用指南

安装依赖

使用uv安装所需依赖：

uv add markdown jinja2

运行脚本

uv run python main.py

配置选项

脚本支持以下配置：

  • content_dir: Markdown文件目录
  • output_dir: HTML输出目录
  • template_file: 模板文件路径
  • title: 页面标题

项目结构

目录说明

  • content/: 存放Markdown源文件，文件名按顺序排列
  • output/: 存放生成的HTML文件
  • templates.html: Jinja2模板文件，定义HTML页面结构
  • main.py: 主脚本，负责读取Markdown文件并生成HTML
  • pyproject.toml: 项目配置文件，包含依赖信息

文件命名规则

Markdown文件建议按照以下规则命名：

XX_filename.md

其中XX为数字，用于控制文件的合并顺序。

模板定制

可以修改templates.html文件来自定义HTML页面的样式和结构：

  • 修改title标签自定义页面标题
  • 修改CSS样式自定义主题
  • 修改section结构自定义内容布局
