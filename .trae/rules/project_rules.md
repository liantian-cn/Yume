# 项目信息

1. 当前工作目录（e:\Documents\GitHub\Yume）是项目目录
2. 项目使用 uv 作为 Python 包管理工具
3. 项目结构：
   - content/：存放markdown源文件（*.md）
   - output/：存放生成的HTML文件
   - templates.html：HTML模板文件
   - main.py：主脚本，用于将markdown生成HTML
   - pyproject.toml：项目配置文件

# 项目用途

将content目录下的多个markdown文件按文件名顺序合并生成一个单页HTML文件，使用Jinja2模板引擎渲染。

# 使用 uv 做 python 的管理

当想执行 main.py 时，执行 `uv run python main.py`

# 使用 powershell

当想执行 main.py 时，执行 `clear; uv run python main.py`
